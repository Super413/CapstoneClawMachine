# CapstoneClawMachine
Claw Machine code for my mini claw machine, designed to run with arduino and nema 17 motors.

# Operating Instructions:
When first initializing machine/program, that position is set as **"zero"**. The **zero** is essentially the position the claw returns to to deposit the prize.

### Gameplay Loop
* Initialize Machine, ensure plugged in to power.
* Navigate claw to just above the prize chute
* Click joystick button to set machine zero (do not click more than once, if not in desired position, either restart machine or press button hidden inside the machine)
* Play! Be mindful of gantry limits, as there are no limit switches.
